Проект представляє собою головну "вступну сторінку" для сервірсу Whitepace, зроблений завдяки шаблону з Figma. 
Адаптивний і оптимізований для різних пристроїв за допомогою media-down. Також для 1920 формату екранів є міксини. Але в цілому медіазапити були виписані вручну, бо багато з секцій сторінки були майже ідентичні і нічим не відрізнялись.

Декоративні SVG-елементи позиціонувались через absolute для візуальної динаміки, та залежно від медіа запиту змінювалися. Охайно організована структура HTML.
